# Train Dataset Exploratory Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on a train dataset using Python.  
The goal is to understand the dataset, clean missing or inconsistent data, and explore meaningful insights using visualizations.

---

## Key Objectives
- Load and explore the train dataset
- Handle missing values and data inconsistencies
- Analyze numerical & categorical features
- Visualize relationships and patterns
- Extract insights for future modeling

---

## Tools & Libraries Used

| Library | Purpose |
|--------|---------|
| Pandas | Data cleaning & manipulation |
| NumPy | Numerical operations |
| Matplotlib | Basic visualizations |
| Seaborn | Advanced visualizations |
